# Indian Market Intelligence Platform

Production-grade real-time stock market analytics and prediction system for Indian markets.

## Features
- Real-time stock data ingestion
- Kafka streaming
- Sentiment analysis
- ML prediction engine
- Real-time alerts
- Dashboard visualization

## Tech Stack
- Python
- Kafka
- PySpark
- FastAPI
- PostgreSQL
- Docker
- AWS
